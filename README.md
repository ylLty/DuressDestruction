## 💡 关于此项目

- 这是一个 [Magisk](https://github.com/topjohnwu/Magisk) 模块，需要使用 Magisk 或者 [KernelSU](https://github.com/tiann/KernelSU) 刷入

- 它可以在您被胁迫交出手机密码的场景下快速销毁您的隐私数据

  > 温馨提示：此模块仅有通过 Magisk 或者是 KernelSU 获取 Root 权限的安卓设备。说您的设备未 Root ，请在网上搜索教程。若您的设备不支持 Root ，目前正在考虑开发一个功能大致相同的 APP ，仅需无障碍与设备管理器权限。~~（画饼中）~~

## 📲 安装方式

- 在此 Release 页面中下载最新版的文件
- 打开 Magisk App 点击右下角的 *模块* ，然后再点击模块页面上方的 *从本地安装* (如图)
- 找到你刚刚下载的文件（一般在下载文件夹中）
- 确认刷入
- 等到 *重启* 字样出现后，点击 *重启* 即可（重启时不要乱按任何按键）

若正常进入系统，请再次打开 Magisk App 并点击右下角的 *模块* ，找到刚刚安装的 *音量键格机 | 胁迫销毁 - Duress Destruction* ，若在描述处显示 [😋 准备就绪!],说明模块加载成功！

## 💥 触发方式

注：

- "+" 表示音量增加键；

- "-" 表示音量减小键；

- "*"表示电源键；

- 加粗部分为触发关键依据

- 锁屏界面指已经亮屏，等待解锁的界面（刚开机、或者是人脸已经识别成功但未解锁的锁屏界面也算）
- +-+- 为判断依据

| 可行性（✅可行；❌不可行；⚠需说明） | 操作             | 时机            | 备注                                                         |
| --------------------------------- | ---------------- | --------------- | ------------------------------------------------------------ |
| ✅⚠                                | **+-+-**         | 开机中/锁屏界面 | 最标准的做法（但有可能检测不到）                             |
| ✅                                 | **+-+-**         | 锁屏界面        |                                                              |
| ✅                                 | **+-+-**+-+-+-+- | 锁屏界面        | **最推荐且稳妥**                                             |
| ✅                                 | -**+-+-**+-+     | 锁屏界面        |                                                              |
| ✅⚠                                | **+-+-**+-+-+-+- | 开机中/锁屏界面 | 未安装  [音量键极速救砖](https://lsposed.cn/105) 时的最推荐做法 |
| ✅⚠                                | -**+-+-**+-+     | 开机中/锁屏界面 |                                                              |
| ❌                                 | +-               | 开机中/锁屏界面 |                                                              |
| ❌                                 | +-*+-            | 开机中/锁屏界面 |                                                              |
| ✅⚠                                | **+-+-** *       | 开机中/锁屏界面 |                                                              |
| ❌                                 | +++-             | 开机中/锁屏界面 |                                                              |
| ❌                                 | --+-             | 开机中/锁屏界面 |                                                              |

>[!WARNING]
>重要说明：所有带有⚠标志的操作方式（即在开机中）都会与 [爱玩机-音量键极速救砖](https://lsposed.cn/105) 产生冲突，因为它的触发条件远比此模块敏感（别看他说要三击，实测不是）
>如果你没有使用这个模块，则可放心操作
>如果安装了，请考虑卸载它，或是仅在锁屏界面触发销毁操作
